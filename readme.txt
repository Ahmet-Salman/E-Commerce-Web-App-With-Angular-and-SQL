- Open the "WebAPI" project in Debugging mode (preferrably using Microsoft Visual Studio)
- Download Angular CLI globally (if not already downloaded) using this command: npm install -g @angular/cli
- Open the "UserInterface" folder in Visual Studio Code, in the terminal, run the server by typing ng serve --open

P.S: to be able to run on your machine, you need to change the database names, the database connection string, and create suitable tables to hold the information
